## 🎯 Lab Objectives & Concepts

1. **Single Page Application (SPA):** Dynamically updates content without full page reloads.
2. **Virtual DOM:** In-memory lightweight representation of real DOM; performs efficient diffing and batch updates.
3. **React Component:** Functional component rendering standard heading: `<h1>welcome to the first session of React</h1>`.

---

## 🚀 How to Run `myfirstreact`

```bash
cd myfirstreact
npm install
npm start
```
Access in browser: `http://localhost:3000`
